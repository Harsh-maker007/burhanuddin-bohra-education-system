Education Platform

Basic demo interface for the education system dataset project.

Dataset Details

This project fills the gap by developing an inclusive prediction and instructional support system. The student-related data includes:

  • % of Attendance
  • % of Hours of Study
  • Previous Scores

The dataset has approximately 100-200 total records.

Data Preprocessing

  • Elimination of gaps in data entry
  • Cleaning of data
  • Standardization of features

Feature Engineering

  • Hours Spent Studying - effort exerted
  • Attendance Rate - amount of class attendance
  • Past Test Scores - periodic means with which to measure progress

Train/Test Split

  • 80% Training
  • 20% Testing
